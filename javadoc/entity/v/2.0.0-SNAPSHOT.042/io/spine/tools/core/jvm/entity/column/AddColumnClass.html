Class AddColumnClass

  • All Implemented Interfaces:

public final class AddColumnClass
extends CreateNestedClass

    Creates a class called Column and nests it under an entity state class.

    The class provides API for obtaining columns for given EntityState type. The Column class is public static and stateless. It serves as a DSL for calling public static methods for obtaining entity state columns.

    Since the Column class is not meant to be instantiated, a private parameterless constructor is generated.

    In addition to methods for obtaining individual columns, a method for obtaining all the columns is also generated.
