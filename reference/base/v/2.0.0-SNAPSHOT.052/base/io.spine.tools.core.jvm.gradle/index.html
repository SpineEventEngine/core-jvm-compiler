base/io.spine.tools.core.jvm.gradle

Package-level declarations

This package provides the Gradle project extension of Spine Model Compiler for Java (mc-java) and associated types.

The extension is the central place of the configuration of mc-java. Therefore, this package is used by modules that comprise mc-java for obtaining corresponding configuration settings.
